## 常见问题解决

- **解决方法**：确保项目文件夹中包含完整的`model/`文件夹，且文件夹中有正确的模型文件

### 3. 背景图片不显示
- **解决方法**：确保已勾选"使用背景图片"复选框，并正确选择了背景图片文件

### 4. 渲染速度慢
- **解决方法**：尝试使用GPU渲染，或选择较低的渲染尺寸

### 5. 中文字符路径错误
- **解决方法**：确保项目路径和皮肤文件路径不包含特殊字符，程序已支持UTF-8编码

## Troubleshooting

### 1. Rendering failed, prompt cannot find Blender path
- **Solution**: Ensure that the Blender path has been set correctly and the path contains the `blender.exe` file

### 2. Rendering failed, prompt cannot find model file
- **Solution**: Ensure that the project folder contains the complete `model/` folder with correct model files

### 3. Background image not displayed
- **Solution**: Ensure that the "Use Background Image" checkbox is checked and the background image file is selected correctly

### 4. Slow rendering speed
- **Solution**: Try using GPU rendering or choose a lower rendering size

### 5. Chinese character path error
- **Solution**: Ensure that the project path and skin file path do not contain special characters, the program already supports UTF-8 encoding
